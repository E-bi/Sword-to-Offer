### 面试题6：从尾到头打印链表
题目：输入一个链表的头结点，从尾到头反过来打印出每个节点的值。
链表节点的定义为：

```
class Node:
    '''
    定义节点类
    data:数据
    _next:下一个数据
    '''
    def __init__(self,data,_next = None):
        self.data = data
        self._next = _next
    def __repr__(self):
        '''
        用来定义Node的字节输出
        '''
        return str(self.data)
```
### 面试题7：重建二叉树

```
class Node:
    def __init__(self,data):
        self.data = data
        self.left = None
        self.right = None
```

### 面试题10：斐波那契数列
题目一：求斐波那契数列的第n项。
写一个函数，输入n，求斐波那契数列的第n项。斐波那契数列的定义如下：
```
f(n)={  
        0                n=0
        1                n=1
        f(n-1)+f(n-2)    n>1
    }
```
